🚀 INICIO RÁPIDO - 3 FORMAS DE USAR:

┌──────────────────────────────────────────────────────────────┐
│ 1️⃣  MODO INTERACTIVO (Recomendado para principiantes)        │
└──────────────────────────────────────────────────────────────┘

  $ python3 generate_cv.py
  
  → Te pedirá que selecciones un color (1-6)
  → Genera automáticamente cv_en_[color].pdf y cv_es_[color].pdf

┌──────────────────────────────────────────────────────────────┐
│ 2️⃣  USANDO MAKE (Rápido y conveniente)                       │
└──────────────────────────────────────────────────────────────┘

  $ make generate         # Modo interactivo
  $ make cv-blue          # Generar con color específico
  $ make cv-red           # Rojo
  $ make cv-green         # Verde
  $ make cv-purple        # Morado
  $ make cv-gray          # Gris
  $ make cv-black         # Negro
  $ make all-colors       # Generar todos los colores
  $ make help             # Ver todos los comandos

┌──────────────────────────────────────────────────────────────┐
│ 3️⃣  AUTOMATIZADO (Para procesos batch)                       │
└──────────────────────────────────────────────────────────────┘

  $ echo "1" | python3 generate_cv.py    # Color azul
  $ echo "2" | python3 generate_cv.py    # Color rojo
  $ ./demo.sh                             # Genera todos los colores

🎨 COLORES DISPONIBLES:

  1. Blue (Azul)      → Profesional y corporativo
  2. Red (Rojo)       → Audaz y llamativo
  3. Green (Verde)    → Fresco y moderno
  4. Purple (Morado)  → Creativo y único
  5. Gray (Gris)      → Minimalista y elegante
  6. Black (Negro)    → Clásico y sobrio
